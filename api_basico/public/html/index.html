Introduction

This API provides endpoints for managing users and inventory items. It allows for creating, reading, updating, and deleting both users and items.

Base URL

All API endpoints are relative to the base URL:

http://localhost:3000/api

Error Handling

The API uses conventional HTTP response codes to indicate the success or failure of an API request.

Status Code Description
200 - OK Everything worked as expected
400 - Bad Request The request was unacceptable, often due to missing required parameters
404 - Not Found The requested resource doesn't exist
500 - Server Error Something went wrong on the server

Get All Users

GET /users

Retrieves a list of all users with their complete item details.

Response

[ { "id": 1, "name": "John Doe", "mail": "john@gmail.com", "items": [ { "id": 1, "name": "Sword", "type": "Weapon", "effect": "Sharpness" }, { "id": 2, "name": "Shield", "type": "Armor", "effect": "Defense" } ] }, // More users... ]

Get User by ID

GET /users/:id

Retrieves a specific user by their ID, including their complete item details.

Parameters

Parameter Type Description
id Integer The unique identifier of the user

Response

{ "id": 1, "name": "John Doe", "mail": "john@gmail.com", "items": [ { "id": 1, "name": "Sword", "type": "Weapon", "effect": "Sharpness" }, { "id": 2, "name": "Shield", "type": "Armor", "effect": "Defense" } ] }

Add Users

POST /users/add

Adds one or more users to the system.

Request Body

An array of user objects with the following properties:

[ { "name": "John", "mail": "john@gmail.com", "items": [1, 2] // Array of item IDs }, // More users... ]

Response

{ "message": "Users added successfully" }

Update User

PUT /users/update/:id

Updates a specific user's information.

Parameters

Parameter Type Description
id Integer The unique identifier of the user to update

Request Body

{ "name": "Updated Name", "mail": "updated@gmail.com", "items": [1, 2, 3] // Array of item IDs }

Response

{ "message": "User updated successfully", "user": { "id": 1, "name": "Updated Name", "mail": "updated@gmail.com", "items": [1, 2, 3] } }

Delete User

DELETE /users/remove/:id

Deletes a specific user by their ID.

Parameters

Parameter Type Description
id Integer The unique identifier of the user

Response

{ "message": "User deleted successfully" }

Get All Items

GET /items

Retrieves a list of all items in the catalog.

Response

[ { "id": 1, "name": "Sword", "type": "Weapon", "effect": "Sharpness" }, { "id": 2, "name": "Shield", "type": "Armor", "effect": "Defense" }, // More items... ]

Get Item by ID

GET /items/find/:id

Retrieves a specific item by its ID.

Parameters

Parameter Type Description
id Integer The unique identifier of the item

Response

{ "id": 1, "name": "Sword", "type": "Weapon", "effect": "Sharpness" }

Add Items

POST /items/add

Adds one or more items to the catalog.

Request Body

An array of item objects with the following properties:

[ { "name": "Axe", "type": "Weapon", "effect": "Cutting" }, // More items... ]

Response

{ "message": "Items added successfully" }

Update Item

PUT /items/update/:id

Updates a specific item's information.

Parameters

Parameter Type Description
id Integer The unique identifier of the item to update

Request Body

{ "name": "Updated Sword", "type": "Legendary Weapon", "effect": "Enhanced Sharpness" }

Response

{ "message": "Item updated successfully!" }

Delete Item

DELETE /items/remove/:id

Deletes a specific item by its ID.

Parameters

Parameter Type Description
id Integer The unique identifier of the item

Response

{ "message": "Item removed successfully" }
