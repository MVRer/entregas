Introducción a CSS

En este post aprenderás qué es CSS, sus componentes básicos y cómo dar estilo a tus páginas web de manera efectiva.

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje que define la presentación visual de documentos HTML, controlando colores, tamaños, posiciones y otros aspectos visuales de los elementos web.

Componentes básicos

Una regla CSS consta de tres elementos fundamentales:

Componente Descripción
Selector Indica qué elemento HTML será estilizado
Propiedad Especifica qué característica se modificará
Valor Define cómo se modificará la propiedad

Selectores en CSS

Los selectores determinan a qué elementos se aplicarán los estilos:

Selector por tipo

p {
    color: blue;
    font-size: 16px;
}

Aplica estilos a todos los elementos del tipo especificado (en este caso, todos los párrafos).

Selector de clase

.destacado {
    background-color: yellow;
    font-weight: bold;
}

Aplica estilos a elementos con la clase especificada: <p class="destacado">

Selector de ID

#cabecera {
    background-color: black;
    color: white;
}

Aplica estilos al elemento con el ID especificado: <div id="cabecera">

Diferencias entre selectores

Tipo Sintaxis Especificidad Uso
Elemento p { } Baja Estilos generales
Clase .clase { } Media Grupos de elementos
ID #id { } Alta Elementos únicos

Propiedades y valores comunes

  • color: red; - Color del texto
  • font-size: 16px; - Tamaño de fuente
  • margin: 10px; - Margen exterior
  • padding: 5px; - Relleno interior
  • background-color: #f0f0f0; - Color de fondo

Formas de incluir CSS

  1. CSS Interno: <style>...</style> en el <head>
  2. CSS en línea: <p style="color: blue;">
  3. CSS Externo: <link rel="stylesheet" href="estilos.css">
