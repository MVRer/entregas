Comandos de terminal

En este post te enseñaré algunos comandos de terminal útiles vistos en clase.

Comandos básicos de Linux

La siguiente tabla muestra los comandos más utilizados en Linux:

Comando Descripción Ejemplos
pwd Muestra la ruta del directorio actual pwd → /home/usuario/documentos
ls Lista los archivos y directorios ls - Lista básica
ls -l - Formato largo
ls -a - Muestra archivos ocultos
cd Cambia de directorio cd documentos - Ir a documentos
cd .. - Ir al directorio padre
cd ~ - Ir al directorio home
mkdir Crea un nuevo directorio mkdir proyecto - Crea directorio "proyecto"
mkdir -p a/b/c - Crea estructura de directorios
touch Crea un archivo vacío o actualiza la fecha de modificación touch archivo.txt - Crea archivo vacío
touch {1..5}.txt - Crea 5 archivos numerados
cp Copia archivos o directorios cp archivo.txt copia.txt
cp -r directorio nuevo_dir - Copia recursivamente
mv Mueve o renombra archivos o directorios mv archivo.txt nuevo.txt - Renombra
mv archivo.txt ~/documentos/ - Mueve
rm Elimina archivos o directorios rm archivo.txt - Elimina archivo
rm -r directorio - Elimina directorio recursivamente
rm -f archivo.txt - Fuerza eliminación sin confirmar
cat Muestra el contenido completo de un archivo cat archivo.txt
cat -n archivo.txt - Muestra con números de línea
less Visualizador de archivos con desplazamiento less archivo.txt - Navega usando flechas o espacio
vi Editor muy básico de archivos vi archivo.txt - La neta todavia no se usarlo
nano Editor más comodo en mi opinion nano archivo.txt - ctrl x para salir y guardar
